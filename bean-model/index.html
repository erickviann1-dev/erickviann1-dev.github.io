基于 IC 分析、覆盖 61 只 A 股 2022–2026 年间 40,292 条日线样本的技术选股模型。6 条核心入场规则 + 3 条追高过滤，分离训练集与测试集验证，测试集表现超过训练集。本页展示 2024-2025 年回测结果，并提供随机抽样实验区域；具体规则阈值与 IC 值不公开。

01
Research Framework

Methodology

Bean Model v2 is an IC-driven multi-factor technical screen built on 40,292 daily observations spanning 61 A-share stocks across 2022–2026. Twenty standard technical indicators — RSI, MACD, Bollinger bandwidth, volume ratio, momentum, and more — were evaluated against forward 10-day returns using Pearson information coefficients.

Indicators were ranked by predictive power, and six core entry rules plus three anti-chase filters were derived from the training set (2023–2024 · 21,143 observations). These rules were then validated independently on the held-out test set (2025+ · 15,073 observations). Out-of-sample performance exceeded in-sample performance — the primary evidence against overfitting.

METHOD · IC DISCOVERY
For each indicator i: IC_i = corr( indicator_i, ret_{t→t+10} ) Rank by |IC|, derive thresholds on 2023–24 training set, validate on 2025+ test set.
Five-step research workflow
  1. Collect daily price-volume data for 61 A-share stocks across 2022-01 to 2026-02 — 40,292 observations
  2. Compute 20 standard technical indicators (RSI, MACD, Bollinger, vol ratio, momentum, etc.)
  3. Calculate Pearson IC of each indicator vs forward 10-day return; rank by predictive power
  4. Derive 6 core entry rules + 3 anti-chase filters from the IC ranking on the training set (2023-2024, 21,143 obs)
  5. Independently validate hit rate and average return on the held-out test set (2025+, 15,073 obs)
⊘
What's withheld The specific thresholds for the 6 rules (e.g., RSI / MACD cut-off points) · IC values for the 20 indicators · Current-period (2026+) live signals and stock picks · Indicator readings on individual signal-trigger dates.
